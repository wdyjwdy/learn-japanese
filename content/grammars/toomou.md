# と思う

## 觉得

> 〔简体〕と思う

```js
とてもよかったと思います。 // 我觉得很好。
```

第三人称使用[〜と思っています](#と思っています)。

```js
彼は私が日本にいると思っています。 // 他一直觉得我在日本。
```

## 想

> 〔意向形〕と思う

```js
出かけようと思います。 // 想出门。
```

否定使用「〜とは思いません」。

```js
一人で行こうとは思いません。 // 不想一个人去。
```

## 一直觉得

> 〔简体〕と思っている

```js
彼は私が日本にいると思っています。 // 他一直觉得我在日本。
```

## 一直想

> 〔意向形〕と思っている

```js
車を買おうと思っています。 // 一直想买车。
```
