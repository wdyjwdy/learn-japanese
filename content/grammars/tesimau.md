# て しまう

## 糟糕

`〔て形〕しまった`

> 宿題を忘れてしまった。#糟糕，忘写作业了。
> 財布を落としてしまった。#糟糕，把钱包弄掉了。

## 动作完成

`〔て形〕しまう` 很少使用「しまった」。

> 宿題は明日出してしまう。#作业明天交。
> ご飯を早く食べてしまいなさい。#快点把饭吃完。
> 本はもう読んでしまった。#书已经看完了。

| 书面语               | 口语缩写           |
| -------------------- | ------------------ |
| 食べ*てしま*います   | 食べ*ちゃ*います   |
| 食べ*てしま*いました | 食べ*ちゃ*いました |
| 食べ*てしま*った     | 食べ*ちゃ*った     |
| 飲ん*でしま*います   | 飲ん*じゃ*います   |
| 飲ん*でしま*いました | 飲ん*じゃ*いました |
| 飲ん*でしま*った     | 飲ん*じゃ*った     |
