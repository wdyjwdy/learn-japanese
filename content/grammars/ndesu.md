# 〜んです

## 请求解释

> 〔简体・名词な・な形な〕んです・の

```js
どこへ行くんですか。 // 去哪里？
どうして遅れたんですか。 // 为什么迟到？
どこへ行くの？ // 去哪里？
どうして遅れたの？ // 为什么迟到？
```

## 表示原因

> 〔简体・名词な・な形な〕んです・んだ

```js
熱があるんです。 // 为什么请假？因为发烧了。
熱があるんだ。 // 为什么请假？因为发烧了。
```

## 表示强调

> 〔简体・名词な・な形な〕んです・んだ

```js
来月結婚するんです。 // 下个月结婚。
明日から日本へ旅行なんだ。 // 明天开始去日本旅游。
```
