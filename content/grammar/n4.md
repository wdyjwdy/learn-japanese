# N4

## ～より～

- _〔名词〕_：表示比较。否定句中使用[～ほど～ません](#ほどません)。

```js
私は犬より猫が好きです。 // 相比狗，我更喜欢猫。
私は何より音楽が好きです。 // 我最喜欢音乐。
```

## ～ほど～ません

- _〔名词〕_：表示比较。肯定句中使用[～より～](#より)。

```js
今は昔ほど楽しくないです。 // 现在没有以前快乐。
```

## ～より～のほう

- _〔名词〕_・_〔名词〕_：两件事物进行对比。

```js
日本語より英語のほうが難しい。 // 英语比日语难。
```

## ～と～とどちら

- _〔名词〕_・_〔名词〕_：两件事物进行对比的提问。通常使用「のほう・どちらも」回答。

```js
東京まで新幹線と電車とどちらが速いですか。 // 到东京新干线快还是电车快？
新幹線のほうが速いです。 // 新干线更快。
どちらもあまり速くありません。 // 都不怎么快。
```

## ～ながら～

- _〔ます形〕_：表示动作同时进行。后者为主要动作。

```js
音楽を聞きながら勉強します。 // 边听歌边学习。
```

## ～ところです

- _〔字典形〕_：即将做。
- _〔ている〕_：正在做。
- _〔た形〕_：刚刚做完。

```js
試合が始まるところです。 // 比赛就要开始了。
牛乳を飲んでいるところです。 // 正在喝牛奶。
ご飯を食べたところです。 // 刚吃完饭。
```

## ～まで～

- _〔名词・字典形〕_：某时刻前的持续性动作。

```js
十二時まで勉強します。 // 学到十二点。
```

## ～までに～

- _〔名词・字典形〕_：某时刻前的瞬间动作。

```js
九時までに寝ます。 // 九点前睡觉。
```

## ～ませんか

- _〔ます形〕_：表示提议。比更[～ましょうか](#ましょうか)委婉。

```js
一緒に食べませんか。 // 一起吃饭吗？
```

## ～ましょうか

- _〔ます形〕_：表示提议。简体使用[意向形](../vocab/transform#意向形)。

```js
帰りましょうか。 // 我们回家吧。
何を食べましょうか。 // 吃什么？
```

## ～ましょう

- _〔ます形〕_：默认提议已经通过。简体使用[意向形](../vocab/transform#意向形)。

```js
行きましょう。 // 我们走吧。
```

## ～ができる

- _〔名词〕_：能够做某事。简体使用[可能形](../vocab/transform#可能形)。
- _〔字典形こと〕_：能够做某事。

```js
日本語ができます。 // 会说日语。
水は飲むことができません。 // 水不能喝。
```

## ～ことがある

- _〔た形〕_：以前做过某事。

```js
日本へ行ったことがあります。 // 去过日本。
一度も海を見たことがない。 // 一次海都没看过。
```

- _〔字典形・ない形〕_：偶尔发生某事。助词有时会用「も」。

```js
時々朝ご飯を食べことがあります。 // 偶尔会吃早饭。
晩御飯を食べないこともあります。 // 不吃晚饭的时候也有。
```

## ～てもいい

- _〔て形・い形くて・な形で・名词で・ない形なく・い形く・な形でなく・名词でなく〕_：表示让步。

```js
狭くてもいいです。 // 小一点也可以。
近なくてもいいです。 // 不那么近也可以。
```

## ～てはいけません

- _〔て形・い形くて・な形で・名词で〕_：表示禁止。

```js
撮っては行けません。 // 不能拍照。
```

## ～なければなりません

- _〔ない形・い形く・な形で・名词で〕_：表示必须。

```js
朝ご飯を食べなければなりません。 // 必须吃早饭。
```

## ～がほしいです

- _〔名词〕_：表示希望。第三者的希望用[～がります](#がります)表示。

```js
車が欲しいです。 // 想要一辆车。
```

## ～たいです

- _〔ます形〕_：表示希望。第三者的希望用[～がります](#がります)表示。

```js
海を見たいです。 // 想看海。
```

## ～がります

- *〔名词〕*＋*〔をほしがります〕*：表示第三者的希望。
- *〔ます形〕*＋*〔たがります〕*：表示第三者的希望。

```js
犬は水を欲しがります。 // 狗想喝水。
誰でも行きたがりません。 // 谁都不想去。
```

## ～といいです

- _〔普通形〕_：表示希望发生的情况。

```js
仕事を見つかるといいですね。 // 能找到工作就好了。
明日は雨が降らないといいなあ。 // 明天不下雨就好了。
```

## ～そうです

- _〔ます形・い形・な形〕_：表示推断。否定句使用「そうもありません」

```js
美味しいそうです。 // 看起来很好吃。
雨は降りそうもありません。 // 看起来不会下雨。
```

## ～がっています

- *〔い形・な形〕*＋*〔がっています〕*：表示第三者的感受。
- *〔い形・な形〕*＋*〔がります〕*：陈述一般倾向。

```js
桜は寂しがっています。 // 小樱感到很伤心。
彼は運動を嫌がります。 // 他很讨厌运动。
```

## ～まま～

- _〔た形・ない形・い形・な形な・名词の〕_：表示状态一直持续。通常导致了不好的后果。

```js
座ったまま寝た。 // 坐着睡着了。
靴のまま家に入る。 // 穿着鞋进房间。
```

## ～から～

- *〔简体・敬体〕*＋*〔から〕*：表示原因。
- *〔简体〕*＋*〔からです〕*：表示原因。

```js
危ないですから、触らないでください。 // 危险，请别碰。
寒いからです。コートを着ています。 // 因为很冷，所以穿着外套。
```

## ～ので～

- _〔简体・な形な・名词な〕_：表示原因。不能接命令句。比[～から～](#から)更正式。

```js
頭が痛いので、今日は休みます。 // 头很痛，今天请假。
```

## ～て～

- _〔て形・い形くて・な形で・名词で・ない形なくて〕_：表示原因或理由，通常用于表达情感。不能接劝诱句。

```js
友達がなくて、寂しい。 // 没有朋友，很寂寞。
```

## ～に～

- _〔动名词・ます形〕_：表示移动的目的。

```js
散歩に行く。 // 去散步。
東京へ遊びに行く。 // 去东京玩。
```

## ～ため～

- _〔动名の・字典形〕_：表示行为的理由。接在意志动词后。

```js
会議に出席するため、日本へ行く。 // 为了开会，去日本。
日本語を勉強するための本。 // 学习日语的书。
```

## ～ために～

- _〔动名の・字典形〕_：表示行为的目的。接在意志动词后。

```js
日本語を勉強するために、日本へ行く。 // 为了学日语，去日本。
```

参考：[ために・ように的区别](../vocab/compare.md#ためにように)

## ～ように～

- _〔ない形・字典形〕_：表示期待的状态。接在非意志动词后。

```js
風が入るように、窓を開けた。 // 为了让风进来，打开了窗户。
```

参考：[ために・ように的区别](../vocab/compare.md#ためにように)

## ～し～し～

- _〔简体〕_：列举多个理由

```js
今日は休みだし、晴れだし、散歩に行こう。 // 今天放假，天气也好，去散步吧。
この牛乳は美味しいし、安いです。 // 这个牛奶又好喝又便宜。
時間もないし、お金もないし、旅行はまた今度にしよう。 // 时间也没有，钱也没有，旅游还是下次吧。
```

## ～たり～たり～

- _〔た形〕_：列举多个动作。末尾通常接「する」。

```js
休みの日は家でアニメを見たり、寝たりする。 // 放假的时候，在家看动漫或睡觉。
何でも食べたり飲んだりできます。 // 能吃能喝。
雨が降ったり、止んだりする。 // 雨时下时停。
```

## ～かもしれません

- _〔简体・な形・名词〕_：表示可能性。

```
今日は雨が降るかもしれません。 // 今天可能会下雨。
```

## ～はずです

- _〔简体・な形な・名词の〕_：根据客观依据，得出比较肯定的推断。

```js
新鮮な魚だから、美味しいはずだ。 // 鱼很新鲜，应该很好吃。
ここから富士山が見えるはずです。 // 这里应该能看到富士山。
```

## ～ようです

- _〔简体・な形な・名词の〕_：个人的推断。

```js
牛乳は古いようです。 // 牛奶好像过期了。
彼は来ないようだ。 // 他好像不会来。
```

## ～みたいです

- _〔简体・な形・名词〕_：个人的推断。正式场合使用[～ようです](#ようです)。

```js
誰か来たみたいだ。 // 好像有人来了。
明日は雪が降るみたいです。 // 明天好像会下雪。
```

## ～なさい

- _〔ます形〕_：表示命令。（教师对学生，父母对孩子）

```js
早く寝なさい。 // 早点睡。
宿題を出しなさい。 // 交作业。
```

## ～ほうがいいです

- _〔た形・ない形〕_：表示劝告。

```js
お酒を飲まないほうがいいです。 // 最好不要喝酒。
野菜を食べたほうがいいよ。 // 菜最好也要吃。
```

## ～ないと～

- _〔ない形〕_：表示催促。如果不做，会导致不好的状况。

```js
ご飯の前に手を洗わないと。 // 饭前必须洗手。
急がないと、間に合わないよ。 // 不快点的话，就来不及了。
```

## ～たら～

- _〔简体た・简体なかった〕_：表示假设。

```js
もし天気がよかったら、散歩に行きましょう。 // 要是天气好，就去散步吧。
```

- _〔た形〕_：表示动作完成之后。

```js
卒業したら、何をしますか。 // 毕业后打算做什么？
```

## ～なら～

- _〔なら形〕_：表示假设。同[～ば～](#ば)。

```js
君なら、きっとできる。 // 是你的话，一定能做到。
```

- _〔简体・な形・名词〕_：表示对事情的判断。

```js
ケーキなら、駅前の店がおいしいですよ。 // 蛋糕的话，车站前那家店很好吃。
```

## ～ば～

- _〔ば形〕_：表示假设。同[～なら～](#なら)。

```js
もし暑ければ、クーラーをつけましょうか。 // 要是热的话，就开空调吧。
```

## ～と～

- _〔简体〕_：表示一定会发生。

```js
ボタンを押すと、切符が出ます。 // 一按按钮就出票。
```

## ～ても～

- _〔简体て形〕_：即使也。

```js
お金がなくても、幸せです。 // 没有钱也很幸福。
何回読んでも、意味がわかりません。 // 读了几遍，还是不懂。
```

## ～のに～

- _〔简体・な形な・名词な〕_：表示意外或遗憾。却。

```js
かばんはまだ新しいのに、もう壊れました。 // 包还是新的，却已经坏了。
雨が降っていないのに、あの人はかさをさしています。 // 没下雨，那人却打着伞。
```

## ～かどうか～

- _〔简体・な形・名词〕かどうか_：表示疑问。
- _〔疑问词〕か_：表示疑问。

```js
答えが正しいかどうかわかりません。 // 不知道答案对不对。
私の誕生日はいつか知っていますか。 // 知道我生日是什么时候吗。
```

## ～と思います

- _〔名字・简体・引用〕_：表示引用。第三人称使用[～と思っています](#と思っています)。

```js
とてもよかったと思います。 // 我觉得很好。
花は英語で何と言いますか。 // 花用英语怎么说。
食事の前に「いただきます」と言います。 // 吃饭前说「いただきます」
```

- _〔意向形〕_：想。否定使用「とは思いません」。

```js
出かけようと思います。 // 想出门。
一人で行こうとは思いません。 // 不想一个人去。
```

## ～と思っています

- _〔名字・简体・引用〕_：表示引用。第一人称使用[～と思います](#と思います)。

```js
彼は私が日本にいると思っています。 // 他以为我一直在日本。
```

- _〔意向形〕_：一直想。

```js
車を買おうと思っています。 // 一直想买车。
```

## ～つもりです

- _〔字典形・ない形〕_：准备。第三人称使用「つもりそうです・つもりらしいです・つもりと言っていました」，否定句使用「つもりはありません・ないつもりです」。

```js
来週は日本に行くつもりです。 // 准备下周去日本。
彼はうちに帰るつもりらしいです。 // 他好像准备回家。
私は自分の意見を変えるつもりはありません。 // 我不准备改变自己的意见。
```

## ～と言っていました

- _〔简体〕_：转述别人说的话。

```js
彼は何と言っていましたか。 // 他说了什么。
彼は最近忙しいと言っていました。 // 他说他最近很忙。
```

## ～そうです

- _〔简体〕_：转述听到的信息。

```js
天気予報によると、今日は雨だそうです。 // 天气预报说今天会下雨。
```

## ～らしいです

- _〔简体・な形・名词〕_：转述从别人那听到的信息。转述自己判断的结果。

```js
聞いた話では、あの店は美味しくないらしいです。 // 听说那家店不好吃。
この店は有名らしいね。よく名前を聞くよ。 // 这家店好像很有名，经常听到名字。
```
