# 动词变形

## 字典形

- **一类动词**

～*U*\
～*AUO*る

- **二类动词**

～*IE*る

- **三类动词**

する\
くる

> 例外：要る・帰る・返る・切る・知る・入る・走る・減る・喋る

## ます形

- **一类动词**

～*U*　→　～*I*ます

- **二类动词**

～る　→　～ます

- **三类动词**

する　→　します\
くる　→　きます

## て形

- **一类动词**

～く　→　～いて\
～ぐ　→　～いで\
～む·ぶ·ぬ　→　～んで\
～う·つ·る　→　～って\
～す　→　～して

- **二类动词**

～IEる　→　～て

- **三类动词**

する　→　して\
くる　→　きて

## た形

将[「て形」](#て形)结尾的「て・で」替换为「た・だ」

## 可能形

- **一类动词**

～*U*　→　～*E*る

- **二类动词**

～る　→　～られる

- **三类动词**

する　→　できる\
くる　→　こられる

## 意向形

- **一类动词**

～*U*　→　～*O*う

- **二类动词**

～る　→　～よう

- **三类动词**

する　→　しょう\
くる　→　こよう

## 被动形

- **一类动词**

～*U*　→　～*A*れる

- **二类动词**

～る　→　～られる

- **三类动词**

する　→　される\
くる　→　こられる

## 使役形

- **一类动词**

～*U*　→　～*A*せる

- **二类动词**

～る　→　～させる

- **三类动词**

する　→　させる\
くる　→　こさせる

## 被动使役形

- **一类动词**

～*U*　→　～*A*される\
～*U*　→　～*A*せられる\
～す　→　～させられる

- **二类动词**

～る　→　～させられる

- **三类动词**

する　→　させられる\
くる　→　こさせられる

## ば形

- **一类动词**

～*U*　→　～*E*ば

- **二类动词**

～る　→　～れば

- **三类动词**

する　→　すれば\
くる　→　くれば

- **い形容词**

～い　→　～ければ

- **ない形**

～ない　→　～なければ

## なら形

- **な形容词**

～　→　～なら

- **名词**

～　→　～なら

- **ない形**

～ない　→　～なければ

## 简体

- **动词**

買う\
買わない\
買った\
買わなかった

- **い形容词**

高い\
高くない\
高かった\
高くなかった

- **な形容词**

便利だ\
便利ではない\
便利だった\
便利ではなかった

- **名词**

雨だ\
雨ではない\
雨だった\
雨ではなかった

## 敬体

- **动词**

買います\
買いません\
買いました\
買いませんでした

- **い形容词**

高いです\
高くないです\
高かったです\
高くなかったです

- **な形容词**

便利です\
便利ではありません\
便利でした\
便利ではありませんでした

- **名词**

雨です\
雨ではありません\
雨でした\
雨ではありませんでした
